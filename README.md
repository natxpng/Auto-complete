# Futebusk ⚽

Aplicação de sugestões inteligentes de busca sobre futebol.

## 🐳 Como rodar o projeto com Docker

### Pré-requisitos
- Git
- Docker

### Passos para rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/natxpng/Auto-complete.git```

2. Acesse a pasta clonada:

   ```bash
   cd Auto-complete
   ```

3. Acesse a pasta raiz do projeto (caso não esteja nela):

4. Rode o projeto com Docker:

   ```bash
   docker compose up --build
   ```

5. Acesse a aplicação em:

   ```
   http://localhost:3000
   ```
